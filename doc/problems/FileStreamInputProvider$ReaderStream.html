ReaderStream

problems.FileStreamInputProvider.ReaderStream
class ReaderStream(val file: File) extends Stream[Char]

Inner implementation of parser.stream.Stream that is returned by the provider (and is not intended to be used directly by the user).

This reads a character whenever needed (i.e. whenever parser.stream.Stream.eat is called). It uses a java.io.BufferedReader internally to improve reading performances.
