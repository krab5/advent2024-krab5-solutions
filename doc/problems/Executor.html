Executor

problems.Executor
final class Executor[Input, Data, Solution](problem: Problem[Input, Data, Solution], input: InputProvider[Unit, Input])

A class that is able to take a problem and an input (as an problems.InputProvider) to effectively setup and run the problem. Executors print a few messages, including which problem is solved and what is its source input, then the result and execution time in seconds.

Value parameters

input

the base input to the problem (used to set up its problems.Problem.inputProvider, given itself as a problems.InputProvider (which is usually more convenient).

problem

the problem to execute
