GridPosition

geometry.GridPosition
See theGridPosition companion object
case class GridPosition[+A](grid: Grid[A], line: Int, column: Int)

A GridPosition is a transient object bound to a grid and specifying coordinates. The idea is to represent the positions of the grid as a first class citizen that can be handled easily, rather than work on the grid itself (which might be cumbersome whenever doing browsing and offseting).

A GridPosition can be turned into the value of the grid at said position, and various iterators may sprout from it.
