GridPositionIterator

geometry.GridPosition.GridPositionIterator
class GridPositionIterator[+A](val grid: Grid[A], val base: (Int, Int), val it: Iterator[(Int, Int)]) extends GridIterator[A]

A GridIterator that enumerates the positions formed by taking base and applying it the offsets given in it, in order.

For example, if the base position is (5,6) and the iterator contains ((1,1),(0,1),(-1,0)), the resulting grid iterator will enumerate the positions ((6,7),(5,7),(4,6)).
