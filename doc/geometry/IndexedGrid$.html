def fromSeqSeq[A](input: Seq[Seq[_ <: A]]): IndexedGrid[A]

Build an IndexedGrid from a sequence of sequences, interpreted as a sequence of lines. The size is guessed by taking the size of the outer sequence (which gives the height of the grid) and the size of the head (which gives the width of the grid). The sequences in the input are assumed to have the same size.
